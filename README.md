<h1>Discover</h1>

<p>
  Este projeto foi desenvolvido durante o curso Discover da Rocketseat,
  com o objetivo de aprender e praticar os fundamentos do desenvolvimento web.
</p>

<p>
  A aplicação consiste em uma página web estática, focada em estruturação
  correta do HTML, estilização com CSS e organização de arquivos.
</p>

<h2>Objetivo do Projeto</h2>

<p>
  O principal objetivo é consolidar conceitos básicos de front-end,
  como semântica HTML, responsividade e boas práticas de código.
</p>

<h2>Tecnologias Utilizadas</h2>

<p>
  O projeto foi desenvolvido utilizando HTML, CSS e Figma,  além do Git e GitHub
  para controle de versão e publicação do código.
</p>
